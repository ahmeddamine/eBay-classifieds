# BazaarLane Marketplace

Modern classifieds marketplace scaffold built with Next.js App Router + TypeScript + Tailwind CSS.

## Quickstart

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Environment variables

Create `.env.local`:

```bash
NEXT_PUBLIC_SUPABASE_URL=your-project-url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key
```
